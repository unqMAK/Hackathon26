# MIT-VPU Hackathon 2025-26

HackSphere - Hackathon Management Platform

## Features
- Team Registration & Management
- Problem Statement Selection (Toggle-based)
- Admin Dashboard with Team Approvals
- Announcement System
- Email Notifications

## Tech Stack
- **Frontend**: React + Vite + TypeScript + Tailwind CSS + shadcn-ui
- **Backend**: Node.js + Express + MongoDB + Mongoose

## Setup

```bash
# Install dependencies
npm install
cd server && npm install

# Start development
npm run dev          # Frontend (port 8082)
cd server && npm run dev  # Backend (port 5000)
```

## Production Build

```bash
# Build frontend
npm run build

# Build backend  
cd server && npm run build
```
